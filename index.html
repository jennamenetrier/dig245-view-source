How to inspect the code on a website


  1. Open the webiste on google chrome
  2. Double click a blank spot on the page
  3. Next, click on Inspect in the option menu that shows up
  4. Finally, the source for the code will be opened in a new window


    5. Looking for the golden egg


    • In the Inspect popup tab (you have to be in Google Chrome), you can edit the code to see how it will look and try things out by selecting the 3 dots on the left hand side of the code and selecting edit with HTML
    • To view a specific code in action, you can find the code you are looking for and right click it. Then you select the tab that says scroll into view and it will take you to the code on the page
    • In the inspect tab, there is a sub banner that says console. If you click on console, you can run javascript code into the system to see how it would look
